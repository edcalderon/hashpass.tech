# Documentation Index

Active docs in this repo are organized by topic. Historical notes, one-off migration writeups, and completed incident docs live in the repository archive at `archive/docs/`.

## Start Here

- [`auth/`](auth/README.md) - production auth and OAuth flows
- [`infra/`](infra/README.md) - environment, API Gateway, Lambda, storage, and security docs
- [`deployment/`](deployment/README.md) - current deployment notes and current runtime decisions
- [`reference/`](reference/README.md) - QR, performance, and release references
- [`storybook/`](storybook/README.md) - Storybook setup, deployment, and guides
- [`guides/`](guides/README.md) - onboarding guides published in Docusaurus and mirrored in Storybook

## Archive

Historical docs, migration notes, and completed incident writeups live in `archive/docs/`.

## Policy

- Keep active docs focused on current behavior and workflows.
- Move completed migrations, one-off fix notes, and incident reports to `archive/docs/` when they are no longer the source of truth.
- Avoid putting secrets, credentials, or pasted production values in docs. Document variable names and flows instead.
